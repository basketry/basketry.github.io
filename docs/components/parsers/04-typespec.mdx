This parser lets you use [TypeSpec](https://typespec.io) as the source for a Basketry project. Add it to your pipeline to generate code, documentation, or apply rules directly from your TypeSpec definition.

## Install

<Install package="@basketry/typespec@alpha" />

## Basic Usage

:::info
This plugin only supports JSON-RPC. Run Basketry with the experimental RPC engine: `npx basketry --engine rpc`
:::

```json title="basketry.config.json"
{
  "source": "petstore.tsp",
  "parser": "basketry-typespec"
}
```
