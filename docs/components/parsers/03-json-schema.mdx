## Install


  </TabItem>
  <TabItem value="yarn" label="yarn">


  </TabItem>
  <TabItem value="pnpm" label="pnpm">


  </TabItem>

## Basic Usage

```json title="basketry.config.json"
{
  "source": "petstore.json",

}
```
